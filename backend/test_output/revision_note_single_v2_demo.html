示例化工有限公司突发环境事件应急预案修编说明 ===================================================================== 一、修编背景 ---------------------------- 根据国家和地方突发环境事件应急管理要求，结合示例化工有限公司的生产经营变化、环境风险源变化及生态环境主管部门的意见反馈，对原《突发环境事件应急预案》进行了修编。 二、修编依据 ---------------------------- 1. 《突发环境事件应急管理办法》 2. 《企业事业单位突发环境事件应急预案编制指南》 3. 《中华人民共和国环境保护法》 4. 《危险化学品安全管理条例》 5. 企业生产工艺、物料及设备变更资料 6. 最近一次专家评审意见（如有） 三、修编原因 ---------------------------- 本次修编主要基于以下原因： - 企业生产工艺或装置发生变动：企业未提供此项信息 - 关键物料变化（危险化学品、危险废物等）：未提供 - 企业组织结构或管理制度更新：未提供 - 上一次预案评审提出的修改意见需要整改 - 相关法规或标准更新 四、修编内容 ---------------------------- 本次修编主要包括以下方面： 1. **风险评估部分更新**：根据最新工艺和物料清单调整风险源信息。 2. **应急资源更新**：更新应急队伍、物资与装备信息。 3. **企业基本信息变更**（地址、法人、产能变化等）。 4. **修订应急响应程序和处置卡**以适应最新风险状况。 5. **新增或修改监测方案**。 6. **专家评审意见整改内容**（见“意见采纳情况表”）。 五、修编过程 ---------------------------- 本次修编由企业成立的预案编制小组组织实施，完成资料收集、风险识别、预案修订、意见征求、内部审核等环节，形成修订版预案。 六、预案实施与更新要求 ---------------------------- 本预案自发布之日起施行。企业应按照实际运行情况，在生产工艺、风险源、应急资源等发生变化时及时修订预案，并至少每三年进行一次评估和更新。 示例化工有限公司 发布日期：2018-03-15
