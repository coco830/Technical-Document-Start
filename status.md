## 📝 开发说明

### 技术栈
- **前端**: Next.js 14 + React 18 + TypeScript
- **UI**: TailwindCSS + Radix UI
- **状态管理**: Zustand
- **后端**: FastAPI + SQLAlchemy (准备就绪)

### 项目结构
```
frontend/
├── src/
│   ├── app/                    # Next.js 页面
│   │   ├── page.tsx           # 首页
```
